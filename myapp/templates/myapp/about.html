About

This tool is designed to help paramedics in the field to diagnose internal bleeding using ultrasound images. Time is of the essence and it is important to accurately determine the situation so that patients can be sent quickly and correctly to a treatment facility.

How to Use

1. Upload the ultrasound images using the Choose File and Upload buttons.

2. Process the image with the algorithm using the Process button.

3. Adjust the variables to create an output image that is the most accurate and clear.
